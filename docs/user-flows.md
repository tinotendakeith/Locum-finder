# User Flows

## Locum Onboarding
1. Register as locum.
2. Verify email (foundation event).
3. Complete profile and upload required documents.
4. Submit profile for review.
5. Admin approves/rejects with reason.
6. Approved locum applies to active jobs.

## Clinic Onboarding
1. Register as clinic.
2. Complete facility profile and upload verification docs.
3. Submit verification.
4. Admin approves/rejects.
5. Approved clinic posts opportunities.

## Job Moderation
1. Clinic creates draft.
2. Clinic submits for review (or auto-publish policy).
3. Admin approves/rejects with note.
4. Active job appears in discovery.

## Application Workflow
1. Locum applies (if approved status and job active).
2. Clinic reviews applicants, shortlists/rejects/accepts.
3. Locum receives notifications.
4. History of status transitions recorded.

## Suspension/Compliance
- Admin can suspend users/profiles/jobs.
- Suspended actors lose protected workflow access.
- Actions are audit logged.
