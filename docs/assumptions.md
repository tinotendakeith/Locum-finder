# Assumptions and Conservative Defaults

- Only approved clinic profiles can publish jobs.
- Only approved locum profiles can apply to jobs.
- Jobs cannot be applied to if not ACTIVE.
- Duplicate applications per job+locum are blocked.
- Admin can override workflow statuses with audit reason.
- List endpoints default to pagination and safe sorting.
- Email/SMS delivery is stub-ready through notification events.
- Sensitive documents are private by default.
- Internal notes and rejection reasons are not publicly visible unless explicitly exposed by role.
