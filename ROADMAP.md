# Delivery Plan

## Phase 1
- Rebuild public landing pages and auth flows to match screenshots.
- Implement locum and clinic profile onboarding.
- Job posting, discovery, application, and application state changes.
- Real-time messaging and notification feed.

## Phase 2
- Admin moderation, verification, and audit workflows.
- Placement confirmations and invoice generation.
- Mobile polishing and push notifications.

## Phase 3
- Billing integrations, payout logic, and analytics.
- Performance tuning and service split decisions.

## Non-goals now
- Microservices split before validated load and team constraints.
- Complex billing before core marketplace traction.
